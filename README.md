# Basic Data Table Manager

A lightweight data table manager built with **Next.js**, **Redux Toolkit**, and **Material UI (MUI)**.  
Supports sorting, global search, pagination, column management, CSV import/export, and inline editing.

## Features
- Dynamic table columns (add, hide/show, reorder)
- CSV import/export (with PapaParse & FileSaver)
- Inline editing with validation
- Global search & pagination
- Theme toggle (Light/Dark mode)
- State persistence with Redux Persist

## Tech Stack
- Next.js 14 (App Router)
- Redux Toolkit
- Material UI (v5)
- TypeScript
- React Hook Form
- PapaParse / FileSaver.js

## Getting Started
```bash
npm install
```
```bash
npm run dev
```
